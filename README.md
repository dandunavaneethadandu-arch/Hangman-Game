# Hangman Game

A simple text-based Hangman Game developed using Python.

## Features
- Random word selection
- 5 predefined words
- 6 incorrect guesses allowed
- Win and Lose conditions

## Concepts Used
- Random Module
- While Loop
- If-Else Statements
- Lists
- Strings

## How to Run
1. Install Python
2. Download hangman.py
3. Run:
   python hangman.py
